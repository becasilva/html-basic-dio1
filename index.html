01. Sobre os H's

As TAGs de H1 até H6 possuem a finalidade de marcar títulos e subtítulos em páginas WEB. O texto fica automaticamente em negrito, em destaque e tem um tamanho bem maior em comparação a um parágrafo convencional.

03. Sobre edição de texto:

Existem algumas maneiras de editar e marcar o texto em HTML.

O deixando em negrito, por exemplo, com a TAG < strong >

Para itálico se usa a TAG < i >

Para sublinhar um texto, usamos a TAG < u >

Para cortar uma palavra ou texto, usamos a TAG < del >

  • Mas também podemos usar a TAG < s > que faz a mesma coisa!

Para marcar o texto como se fosse com marcador de texto se usa a TAG < mark >

Para elevar um texto, como em potências, na matemática, se usa a TAG < sup >

Em complemento à TAG anterior, para abaixar um texto, se usa a TAG < sub >

A TAG < small > deixa o texto pequeno!

A TAG < font > nos permite editar o estilo de um texto. Alterando a fonte, a cor e o tamanho!

E pra finalizar, a minha preferida é a TAG < blockquote >, pois ela apresenta um molde de citação! Olha só:

- "Haja luz"

04. Sobre Listas:

Uma forma muito legal de listar coisas em HTML, é usando a TAG < ol > do inglês ordered list ou seja, Lista Ordenada, em português. E a TAG < ul >, do inglês, unordered list, traduzindo-se para Lista Não-Ordenada.

Os itens das listas ordenadas e não ordenadas são marcadas pela TAG < li > de list item, em inglês. E se apresentam da maneira a seguir:

Ordenadas:

  1. Item 01
  2. Item 03
  3. Item 03

Não ordenadas:

  • Item 01
  • Item 03
  • Item 03
